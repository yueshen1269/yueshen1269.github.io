React中setState的使用

1.基本用法

// 形如：a: 1 -> a: 2
this.setState({key: value}),将传入的对象浅合并到state中，如果存在key，则替换，否则新增
// modal: {visible: false,message: “show this”},更新visible: true
this.setState(key: Object.assign({}, this.state.obj, {key: value}))
或者
this.setState(key: {…this.state.key,…{key: value}})

2.更新

setState可能不会立即更新，如果需要拿到更新后的值

  1. this.setState({key: newValue},() => {
    console.log(this.state.key)}),传入一个回调函数
  2. 在componentDidUpdate(prevProps, prevState) 生命周期内获取

3.保证更新

如果后续状态取决于当前状态，setState()第一个参数传入函数
this.setState((preState,preProps) => {
return {key: value}
})

4.误区

this.forceUpdate()并不能让setState()立即更新，只是重新渲染，但state的值不一定会被更新
