如何减少 js 代码中 if-else 的使用及优化

写代码记住三个字即可，短简易

1. 看能否使用 || 或 && 代替简写

2. 使用三目运算符

3. 使用哈希表

后台接口通常会返回这种数据：
fruit: 0 // 0=苹果，1=梨子，2=桔子，3=柠檬，4=芒果…
这时写 if…else 是最痛苦的

4. 优化 if 逻辑

把最可能发生的排在前面，依次降低

5. 少使用 else

使用if+return判断错误条件，防止进入 else 分支

6. 判断数量多，可以考虑使用 switch/case

  1. case 判断使用的是 === 而不是 ==
  2. 不要忘记在每个 case 语句后面放一个 break 语句，或者 return、throw 语句

7. hash 表

8. 重构，用 OO 里面的继承或者组合

9. 实例

成长速度为 12 显示 3 个箭头；
成长速度为 15 显示 4 个箭头；
其他都显示都显示 0 各箭头。
